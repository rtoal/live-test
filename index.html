Practice your TDD

Begin by writing a test. A test is a block of JavaScript ending with an expression, which is the test’s assertion. You’ll see a red bar on the left of the test because it is (or should be) failing. Next, write some code to make the test pass—to get that green bar.

Tests are live (run continuously). Navigate between the test panel and editor with Ctrl+,. Navigate among the tests with Tab and Shift+Tab.

Add more tests by clicking the ⊕ button under the last test or pressing Shift+Enter in a test box.

Each test is independent. If you need some code to run before each test (e.g., data structure initialization), enter it in the “Before” box.

If the first line of a test is a comment, it will be used in the exported test script as the test’s description.
